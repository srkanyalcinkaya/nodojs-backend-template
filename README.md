# NodeJS Backend Template

```
git clone https://github.com/srkanyalcinkaya/nodojs-backend-template.git
```

```
cd nodojs-backend-template
```

```
npm install 
or
bun install
```

Edit .env.example and start backend

!development
```
bun dev 
or
npm run dev
```
!production
```
bun run start
or
npm run start
```
